##docker

介绍

docker是一个开源的应用容器引擎，给予go语言并遵从Apache2协议开源。

详细介绍

安装

deepin在wiki上有安装教程

常用命令

查找镜像 : docker search ubuntu

下载镜像 : docker pull ubuntu

本机和docker容器之间文件的上传和下载

文件上传到docker：docker cp 本地路径文件 ID全称:容器路径

文件下载到本地：docker cp ID全称:容器文件路径 本地路径

进入docker容器中（shell）

docker exec -it [ID] /bin/bash

docker容器的开始/暂停/停止/删除

开始：docker start CONTAINER

停止：docker stop CONTAINER

暂停：docker pause CONTAINER

开始暂停：docker unpause CONTAINER

删除：docker rm CONTAINER

docker端口映射

自动端口映射

随机映射到本机的49000~49900
docker run -P image_name

指定端口映射

docker run -p 127.0.0.1:8080:80 -d image_name

查看端口映射

docker port CONTAINER [port]
