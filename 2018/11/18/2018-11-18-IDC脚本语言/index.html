IDC脚本语言

介绍

IDC语言是一种跟C语言比较类似的脚本语言，它是一种解释性语言，它是可以运行在IDA中的，IDA可以加载另一种语言就是python

基础知识

执行脚本的方法：

  1. file->script file 可选的文件类型有”.idc”、”.py”
  2. file->script command 在这个里面可以选择语言的种类，IDC或者是python，如下图所示

  1. 如果要执行单个语句的话可以在IDA窗口下方有一个输入框，可以直接执行并且有回显，但如果你要在这里执行多个语句也不是不行的，每个语句只要用分号隔开就行啦，如下图所示：

变量

在IDC中变量没有明确的类型。IDC使用三种数据类型：整形，字符串，浮点型，在使用变量前，都必须先声明该变量，并有局部变量和全局变量之分

局部变量在声明的时候可以赋值，但是全局变量在声明的时候不可以赋值。并且全局变量只保存第一次赋值的值，局部变量在函数返回时销毁，全局变量在关闭IDA数据库时销毁。

注释

IDC中可以使用”/**/“来进行多行注释，以及使用//的C++风格行尾注释。

IDC表达式

在IDC中，支持C中的大多数算术运算符和逻辑运算符，包括三元运算符、逗号运算符。所有整数操作数均作为有符号的值来处理，那么如果运算中涉及到移位的话一定要注意。字符串赋值将导致字符串复制操作，因此你不需要使用字符串复制函数；将两个字符串相加就可以将两个字符串进行拼接，这个规则和python是一样的，也不需要函数来处理；IDC也支持字符串的分片，分片就是从给定的字符串中分离出部分内容。这个规则跟python也有点相似，如下所示：

IDC语句

和C语言一样，IDC中所有的语句都是以分号结束的。同样支持循环，判断等语法结构，但是不支持switch语句；由于IDC不支持复合赋值运算，那么，for语句就稍微的有一些变化：

除此之外，IDC中还引入了try/catch块和相关的throw语句，我在异常处理这块也没仔细的研究过，所以这里我不做过多解释啦。

IDC函数

IDC仅仅在独立程序(.idc,.py)中支持用户定义的函数。在IDC命令对话框不支持用户定义的函数。在这个脚本语言中定义函数是与C语言定义函数有着巨大的差别的，在IDC中，static关键字用于引用一个用户定义的函数，函数的参数列表仅包含一个以逗号分隔的参数名列表，如下所示：

，另外，IDC中同样支持函数的参数按值传值、按地址传值，不同的是，指明按值还是按地址不是在函数定义的时候，而是在函数调用的时候指出的

有用的IDC函数

想要操作IDA数据库，我们就知道IDC语法和规则是不够的，接下来我总结一下IDC中内置的函数

  • 读取数据的函数
    • long Byte(long addr)–>从虚拟地址addr读取一个字节值
    • long Word(long addr)–>从虚拟地址读取一个子(2字节)值
    • long Dword(long addr)–>从虚拟地址读取一个双字(4字节值)
    • bool
  • 修改数据的函数
    • void PatchByte(long addr, long val)–>设置虚拟地址处的一个字节值
    • void PatchWord(long addr, long val)–>设置虚拟地址处的一个字值
    • void PatchDword(long addr, long val)–>设置虚拟地址处的一个双字值

使用这些函数的时候，每一个都考虑到了当前处理器模块的字节顺序(小端或大端)。

如果使用读取数据库的函数时传入一个无效的虚拟地址，则分别会返回0xff,0xffff,0xffffff。但是当我们的数据库值为0xff……时我们就分不清是无效的地址还是真实值，那么我们就在读取数据库之前调用isLoaded函数，以确定判断这个地址是否包含任何数据

  • 用户交互函数

    • void Message(string format)–>在输出窗口打印一条格式化消息，这个函数和C语言中printf函数有点类似，接受printf风格的格式化字符串

    • void print(…)–>在输出窗口打印每个参数的字符串表示形式,如果参数是数字，则会显示出对应的十进制、十六进制、二进制、ISCII，若输入字符串，则会显示字符串本身

    • void Warning(string format)–>在对话框中显示一条格式化消息，这个函数的意思也就是将用户想要输出来的东西用弹框显示出来，相当于HTML中的alert。

    • string AskStr(string default,string prompt)–>显示一个输入框，要求用户输入一个字符串值，如果操作成功则返回用户输入的字符串，如果对话框被取消，则返回0

    • string AskFile(long doSave,string mask,string prompt)–>显示一个文件选择对话框，以简化选择文件的任务。可以创建新文件保存数据(doSave=1)，或者选区现有的文件进行读取数据(doSave=0)。可以根据mask(“*.*或者*.idc”)过滤显示的文件列表。如果操作成功，则返回选定文件的名字，如果对话框被取消，则返回0

    • long AskYN()–>用一个答案为是或者不是的问题来提示用户，参数一是默认的答案，是返回1，否返回0，取消返回-1；

    • long ScreenEA()–>返回当前光标所在位置的虚拟地址

    • bool Jump(long addr)–>跳转到反汇编窗口的指定位置

  • 字符串操纵函数

    • string form(string format,string val)–>返回一个新字符串，该字符串根据所提供的格式化字符串和值进行格式化。

    • string sprintf(srting fromat)–>代替form

    • long atol(string val)–>将十进制值val转换成对应的整数值,我认为这个函数是将十进制转换成ASCii码，emmmmmmmm

    • long xtol(string val)–>将十六进制的val转换成对应的整数值;同上

    • string ltoa(long val,long radix)–>以指定的radix(2、8、10、16)返回val的字符串值
    • long ord(string ch)–>返回单字符串ch的ASCII值
    • long strlen(string str)–>返回str的长度
    • long strstr(string str,string substr)–>返回str中substr的索引。若没有，则返回-1
    • string substr(string str,long start,long end)—>对字符串进行分片

  • 文件输入/输出函数
    对于生成大量文本或者二进制数据的脚本，我们就可以将我们的输出存到本地的文件上面，上面我提到关于文件的函数，AskFile()，这个函数仅仅返回一个包含文件名的字符串值，但是这个函数并不能满足我们的要求。

    • long fopen(string filename, string mode)–>返回一个文件句柄(如果发生错误，则返回0)，供所用IDC文件输入/输出函数使用，第二个参数和C语言中使用的模式是一眼过的，r表示读取，w表示写入，等等。
    • void close(long handle)–>关闭fopen中文件句柄指定的文件

    • long filelength(long handle)–>返回指定文件的长度，如果发生错误，则返回-1

    • long fgetc(long handle)–>从指定文件中读取一个字节，如果发生错误，则返回-1

    • long fputc(long val,long handle)–>想指定的文件中写入一个字节，操作成功，返回0，操作失败，返回-1

  • long fprintf(long handle,string format…)–>将一个格式化字符串写入到文件中

  • long writestr(long handle,string str)–>将字符串写入到指定的文件中

  • string/long readstr(long handle)–>从给定的文件中读取一个字符串，这个函数读取到下一个换行符为止的所有字符，如果操作成功则返回字符串，如果读取到文件尾部，则返回-1。
  • long writelong(long handle,long val,long bigendian)–>使用大端(bigendian=1)或者是小端(bigendian=0)字节顺序将一个4字节整数写入到给定文件中。
  • long readlong(long handle,long bigendian)–>使用大端(bigendian=1)或者是小端(bigendian=0)字节顺序从给定的文件中读取一个4字节整数
  • long writeshort(long handle,long val,long bigendian)–>使用大端(bigendian=1)或者是小端(bigendian=0)字节顺序将一个2字节整数写入到给定文件中。
  • long readshort(long handle,long bigendian)–>使用大端(bigendian=1)或者是小端(bigendian=0)字节顺序从给定的文件中读取一个2字节整数
  • bool loadfile(long handle,long pos,long addr,long length)–>从给定的文件的pos位置读取length数量的字节，并将这些字节写入到以addr开头的数据库中
  • bool savefile(long handle,long pos,long addr,long length)–>将以addr数据库地址开头的length数量的字节写入给定的文件的pos位置
